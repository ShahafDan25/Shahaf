The Market's Manager

See Code  

March 2020


The Market is a monthly service, ran and managed by the Las Positas College Student Government [LPCSG]. In the event, the student government collaborates with the Alameda County's Food Bank to distribute free groceries to the students and anyone from the surrounding communities.

As I volunteer monthly in the event, I noticed the management system is extremely inefficient and time consuming, mainly to the lack of automation and technological optimization.

As a young software developer, I develop a manegement web-app on a closed server using HTML, JavaScript, CSS, and Bootstrap API's for the front end user interface, and PHP for the back end with embedded MySQL queries in accordance to the designated database I created.

The software has significantly improved the management of the service. It provides the users with tools such as sign up sheets for volunteers (and user registration), admin report generation for each market both in PDF fprmat (using FPDF library) and in HTML format with graphics to illustrate statistics (using Morris.js), the option to manage effectively the inventory of each market occurence, and a user-friendly registration form.

Additionally, the software assists with the analysis of data to increase the rates of attendance, describe when volunteers are needed the most, and more.

BrainStorm

See Code  

June 2020


---- PROJECT UNDER CONSTRUCTIONS AT THE MOMENT ----

BrainStorm is an inter-collegiate online tool specified for community college student, with the simple objective of helping students navigate a better and more effective path towards their transfer to a 4 year institution.

It provides students with the opportunity to compare classes from different community colleges (based on a location and specified radius) which will allow them to choose the best class that would fit their learning style from a variety of community coleges.

The site was born from the need to carefully select classes in an era where online-learning is incredibly significant and powerful. Student will have the oppotunity to ask difference students from other community colleges as well as alumnis about their transfer experience, and what academic path do they recommend the most.
